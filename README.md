Portal Estudiantil es una aplicación web diseñada para que los estudiantes tomen el control total de su vida académica. Olvídate de perder fechas de exámenes o traspapelar apuntes; centraliza todo en una interfaz intuitiva y moderna.

-Gestión de Tareas y Proyectos: Anota, organiza y haz seguimiento de tus pendientes diarios.

-Calendario de Exámenes: Agenda tus fechas importantes y recibe recordatorios.

-Central de Apuntes: Espacio dedicado para guardar y organizar tus notas de clase.

-Asistente con IA: * Automatización de Tareas: Función inteligente para ayudar en la resolución de actividades.

Frontend: HTML5, CSS3, JavaScript.

Inteligencia Artificial: Groq

Despliegue: GitHub Pages
